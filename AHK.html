Auto Hotkey

What is Auto Hotkey?
Auto Hotkey, AHK for short, is a scripting language commonly used to make macros.
These macros can range from making alternate key bindings to giving faster than human input for task automation.
In order to use the scripts I have provided here you will need the Auto Hotkey software.
If you would like to learn more about it here are some useful links.

Menu Quick Access

This script is meant to be used for 'Grand Theft Auto 5: Online'
The purpose of this script is to quickly iterate through some of the common in game menus.
Some examples of this are the phone contacts like the mechanic, or even the CEO abilities in the interaction menu.
The script features a very simple GUI to allow users without much coding background a chance to use.
The script will generate a .ini file upon first use if one does not exist for it.
SIDE NOTE 1: As this game is constantly being updated I cannot guarantee that the functionality of the script will remain intact.
SIDE NOTE 2: AHK scripts are 100% legal to use in the game and will NOT result in a ban.
