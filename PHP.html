PHP

Hypertext Preprocessor, also known as PHP,
is a server side scripting language used in back end web development.
It is commonly used in tangent with SQL to access data within a database that is correlated with a website.

Race Horse Database Website

A website that is mainly supported by PHP that offers the three primary options
to the user. Add a new horse(along with its breeders), Add a new race, and Delete a race.

PHP: Server Connection / Data Viewer / Add a horse / Add a race / Delete a race
